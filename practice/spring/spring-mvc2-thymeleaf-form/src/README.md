# Section 03 - 폼 바인딩 동작 흐름

> 이 섹션의 핵심은  
> HTML Form과 자바 객체가 어떻게 연결되는지 이해하는 것이다.

---

# 1️⃣ 폼 화면 요청 (GET)

Controller  
↓  
빈 객체 생성  
↓  
Model에 저장  
↓  
View 반환  
↓  
th:object로 폼과 연결

---

# 2️⃣ 화면 렌더링

th:object  
→ 기준 객체 설정

th:field  
→ id, name 자동 생성  
→ value 자동 세팅

📌 렌더링 시 기존 값이 자동 반영된다.

---

# 3️⃣ 폼 제출 (POST)

사용자 입력  
↓  
HTTP 요청 파라미터 생성  
↓  
@ModelAttribute  
↓  
객체 생성 및 필드 바인딩

📌 ArgumentResolver가 내부에서 처리

---

# 4️⃣ 체크박스 동작 원리

## 문제

체크 해제 시 값이 아예 전송되지 않음

## 해결

hidden 필드를 함께 전송하여  
항상 값이 넘어가도록 처리

---

# 5️⃣ 다중 체크박스 흐름

여러 값 선택  
↓  
동일한 name으로 여러 값 전송  
↓  
List 또는 배열로 바인딩

---

# 6️⃣ 라디오 버튼 흐름

하나만 선택  
↓  
하나의 값만 전송  
↓  
단일 필드에 바인딩

---

# 7️⃣ 셀렉트 박스 흐름

선택된 value 값 전송  
↓  
객체 필드에 바인딩

multiple 사용 시  
→ 컬렉션 바인딩

---

# 8️⃣ 전체 동작 구조 정리

HTML Form  
↓  
name 속성  
↓  
HTTP 요청 파라미터  
↓  
@ModelAttribute  
↓  
객체 필드 자동 바인딩

---
